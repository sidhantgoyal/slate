Customize BuyNow button

You can customize the BuyNow button size and color using the options shown in the following table.

SizeOrange - Default ColorBlue Color
Primary - Default
Compact
Mini

  • Objective_C:

    // Change Size to Compact Scheme
buyNowButton.sizeScheme = WMTButtonSizeSchemeCompact;

// Change Color to Blue Scheme
buyNowButton.colorScheme = WMTButtonColorSchemeBlue;

  • Swift:

    // Change Size to Compact Scheme
buyNowButton.sizeScheme = WMTButtonSizeScheme.Compact;

// Change Color to Blue Scheme
buyNowButton.colorScheme = WMTButtonColorScheme.Blue;

Once you have created a BuyNow button it is also possible to add or remove associated productIds. See the code example below:

  • Objective_C:

    // Add a single product
[buyNowButton addProductId:{productId}];

// Or Remove a single product
[buyNowButton removeProductId:{productId}];

// Or you can also Set Products
[buyNowButton setProductIds:{setOfProductIds}];

  • Swift:

    // Add a single product
buyNowButton.addProductId(productId)

// Or Remove a single product
buyNowButton.removeProductId(productId);

// Or you can also Set Products
buyNowButton.setProductIds(setOfProductIds)
